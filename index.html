Osimo-BBCode-Editor by meltingice

An easy to use jQuery plugin BBCode editor

Dependencies

jQuery >= 1.3.x

Install

Installing the Osimo BBCode Editor is a very simple process and only requires a few steps. First, we need to include the Javascript for the editor. Make sure you include jQuery before you include the osimo_editor.js file! You can do this either in the head section of the html or later in the page.
